Three axis neutron spectrometer with flat-cone analyzer arrangement.

The analyzer is put in flat-cone geometry by setting properties in the Instrument setup / Interface dialog. Set option flat-cone analyzer = on/off.

Set option Q off-plane = on to choose the mode with tilted flatcone: keeps AB plane in the horizontal position if Q is out of this plane. Otherwise, the sample is tilted to keep Q horizontal.

Use the commands in the script TAS_flatcone.inp to try this example.

Remarks:
Use the commands in the attached *.inp script to perform various settings and commands.
You can open the script in the Tools/Script editor and run selected blocks (select text by mouse, then "Run selected"). See the comments at the command blocks in the scripts for brief explanations.
