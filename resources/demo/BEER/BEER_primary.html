ToF diffractometer BEER@ESS.
Primary diffractometer (source ... sample).

Remarks:
Use commands in the attached *.inp scripts to perform more complex settings.
You can open the script in the Tools/Script window and run selected blocks (select text by mouse, then "Run selected").

BEER_setup.inp Setting of neutron optics geometry, distances etc.
modes_setting.inp Setting of various operation modes (see comments in the script)

Other configurations:

BEER_Fe.xml Complete diffractometer with duplex steel sample. This configuration corresponds to the medium resolution pulse-shaping mode (single frame). Hint: run simulation and view results (plot command DET1D).
BEER_MCPL.xml Primary diffractometer with a monitor at 50 mm in front of the sample position. It permits to export neutrons to an MCPL file for subsequent simulation of the sample and secondary part by another program (tested with McStas).
After simulation, you can export neutrons from the selected monitor to MCPL by typing a command in the console window or in the Tools/Script window, e.g.:
MCPL file.mcpl
